# Django DRF Projekt

Dieses Projekt wurde mit Django und Django REST Framework initialisiert.

## Starten des Projekts

1. Installiere die Abhängigkeiten (bereits installiert):
   - django
   - djangorestframework

2. Starte den lokalen Server:
   
   ```powershell
   C:/Users/Phil/AppData/Local/Programs/Python/Python313/python.exe manage.py runserver
   ```

3. Die API-App befindet sich im Verzeichnis `api`.

## Hinweise
- Weitere Apps und Endpunkte können über die Django-typische Struktur hinzugefügt werden.
- Die Konfiguration für DRF kann in den `settings.py` vorgenommen werden.
